Editor

 

The editor window is divided in four parts :

  • On the rigth (Buttons), you can selected the button you want to program.
  • On the left (Configuration), you can change the configuration of the button you have selected. Depending of the button you will see diferent options :
    • Assign to port : asign the current button/hat/axis to the selected analog gameport button/POV/axis.
    • Auto-repeat : activate key repeat emulation on the button selected.
    • Press/Release combos : select the command to assign to the current button (see command editor below). Combos on the right are Pinkie activated.
    • Increment/Position : For axis and rotaries mode of operation.
    • Command/Positions lists : To add a command (and a position) select the command on the Command list and then press on the "+" button. To delete a command (and a position) select the command on the Positions list and press on the "-" button. Lists on the right are Pinkie activated.
  • On the bottom (Special functions), you can de/activate the two "special" modes available for the X36 and X35, these are, the mode switch function and the Pinkie switch function.
  • On the top, you can see three buttons (from left to right) : Command Editor, Save, Report/Print
    • Command editor button opens the command editor window.
    • Save button saves your the current work to disk. The configuration is saved automatically on exit too.
    • Report/Print button has two purpuses : once clicked you´ll see a briefing with all the buttons, rotaries and axis; and then , if you press the print button, you can print that briefing.
