Command window

 

This windows is used to configure the commands that will be used later in the editor.

In Name you must indicate the name of the command.

To add a key to the command, click on the Pulsation text box and press the key. Once pressed you´ll see it´s name on the text box and then you can press on the "<" button to add the key to the key list.
...The ">" button deletes the selected key from the list.
