Main window( Launcher )

 

The main window is used to load the key configuration (keymap) on the driver. The buttons on this screen are these:

  • Load : Loads the key configuration (program joysticks) selected in the list. Once loaded it is not necesary to keep X36Map opened.  
  • New : Creates a new empty keymap.
  • Edit : Edits the selected keymap.
  • Import : Imports X36Laucher profiles (.X36,.CMM files). In order to import a profile .X36 and .CMM files must be in the same directory.
  • Update : Updates keymap list.
  • Exit : ...

Furthermore in the mouse configuration section there are these controls :

  • Refresh rate and precision : With these values the mouse movement sensibility is controlled. *The refresh rate has influence on the CPU, lower values requieres more CPU usage.
  • Update : Sends the new configuration data to the mouse driver.
