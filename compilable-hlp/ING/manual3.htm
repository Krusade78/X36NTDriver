Command editor

 

A command is simply an "alias" for a key combination like Shift + F1, Alt + A , Z , etc. This is used because a game (specially simulators) can have many key combinations and it is easier to manage command names than key combinations.

In the command editor window you can see a list with two columns, one with the command names and another with the keys asociated to the commands.

There are also four buttons in the window. With the first three you can Add, Edit or Delete a command and with Add Mouse you can add mouse emulation to the joystick.

If you click on Add or Edit you´ll see a new screen (the Command window) to configure the commands and the keys they have asociated.

The fourth button ( Add mouse ) adds six special commands without keys. These commands represents the mouse movements and the left and right mouse buttons.
Once added these commands can be used to emulate the mouse with joystick buttons. The commands work as the normal commands but they cannot be edited.
